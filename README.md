# FakeStore E-Commerce App

A responsive e-commerce front-end built with React that interacts with the FakeStoreAPI.  
Users can browse products, view product details, and perform create, update, and delete operations.

> Note: FakeStoreAPI is a mock API. POST, PUT, and DELETE requests return successful responses but do not permanently modify the data.

---

## Features

- Home page with introduction
- Product listing page
- Product details page
- Add product functionality (POST)
- Edit product functionality (PUT)
- Delete product functionality (DELETE)
- Confirmation modal before deletion
- Loading indicators
- Error handling
- Responsive design using React Bootstrap
- React Router for multi-page navigation

---

## Technologies Used

- React
- React Router DOM
- Axios
- React Bootstrap
- Bootstrap 5
- Vite
